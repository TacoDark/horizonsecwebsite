# HorizonSec Website

A modern, minimalistic website for HorizonSec - an Arch-based Linux penetration testing OS.

## Features

- **Greek-Inspired Design**: Classical aesthetic with modern web design principles
- **Color Palette**: Orange, yellow, grey, and brown tones
- **Smooth Animations**: Scroll-triggered animations, parallax effects, and interactive elements
- **Responsive**: Fully responsive design that works on all devices
- **Contributors Section**: Showcases the team behind HorizonSec

## Design Elements

- Gradient backgrounds and text effects
- Greek decorative patterns
- Animated terminal preview
- Interactive feature cards
- Profile avatars with rotating borders
- Smooth scroll navigation

## Structure

- `index.html` - Main HTML structure
- `styles.css` - Complete styling with Greek-inspired design
- `script.js` - Interactive JavaScript with animations
- `pfp/` - Profile pictures for contributors
