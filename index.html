About HorizonSec

HorizonSec was created by security professionals, for security professionals. We recognized the need for a penetration testing distribution that combines the power and flexibility of Arch Linux with a curated selection of security tools.

Unlike bloated alternatives, HorizonSec gives you exactly what you need, nothing more, nothing less. Whether you're conducting network assessments, web application testing, or wireless security audits, HorizonSec provides a solid foundation to build upon.
